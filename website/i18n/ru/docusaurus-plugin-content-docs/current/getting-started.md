# Быстрый старт

## Установка

Все способы (скрипт установки, Scoop, архивы релизов и сборка из исходников) —
на странице [Установка](installation.md).

## Создание Takefile

```sh
taku init          # создаёт стартовый Takufile.lua (существующий не перезапишет)
```

## Команды

| Команда | Алиас | Описание |
|---|---|---|
| `taku run <task>` | `r` | Выполнить задачу и её зависимости |
| `taku list` | `ls` | Показать список задач |
| `taku init` | | Создать стартовый `Takufile.lua` |
