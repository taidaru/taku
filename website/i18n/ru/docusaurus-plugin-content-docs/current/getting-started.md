# Быстрый старт

## Установка

Все способы (скрипты установки, Homebrew, Scoop, архивы релизов и сборка из
исходников) — на странице [Установка](installation.md).

## Создание Takufile

```sh
taku init          # создаёт стартовый Takufile.lua (существующий не перезапишет)
```

## Команды

| Команда | Алиас | Описание |
|---|---|---|
| `taku run <task>` | `r` | Выполнить задачу и её зависимости |
| `taku list` | `ls` | Показать список задач |
| `taku init` | | Создать стартовый `Takufile.lua` |

Запуск `taku` без команды печатает справку. `Takufile.lua` читается только из
текущего каталога (Taku не ищет в родительских каталогах).

Независимые задачи выполняются параллельно — вплоть до числа ядер CPU. Ограничить
число одновременных задач можно флагом `-j` / `--jobs`:

```sh
taku run ci -j 2
```
