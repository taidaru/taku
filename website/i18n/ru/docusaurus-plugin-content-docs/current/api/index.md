# API

Внутри Takefile доступны эти глобалы, плюс библиотеки Lua `string`, `table`,
`math`, `utf8` и `coroutine`:

| Глобал | Назначение |
|---|---|
| [`sh`](./sh.md) | запуск команд |
| [`fs`](./fs.md) | файловая система |
| [`net`](./net.md) | TCP / HTTP(S) |
| [`env`](./env.md) | переменные окружения |
| [`ssh`](./ssh.md) | работа на удалённом хосте |
| `task`, `import` | определение задач, подключение файлов |

Всё остальное (`io`, `os`, `package`, `debug`, ...) отключено — см.
[Песочницу](../sandbox.md).

Пути — это строки; содержимое файлов и вывод команд — **байтовые строки**
(безопасны для бинарных данных). При ошибке функции бросают Lua-ошибку с
контекстом, которую можно перехватить через `pcall`.
